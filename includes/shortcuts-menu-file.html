File Menu Shortcuts

Command Description  
New create a new data table ⌘ N
Open Open, Comma Separated Value file Ctrl O ⌘ O
Save Save the data in the current Tab in the same format, name, and location. Ctrl S ⌘ S
Save As Save the data in the current Tab using a different format, name, or location. Shift Ctrl S ⇧ ⌘ S
