Export data package

You can open an existing data file to validate and improve it. As the file is opened Data Curator guesses the validation rules for each column. You can validate the data using these rules or refine them. Data Curator also checks for ragged rows on opening the file.

Export data package

To export a data package, either:

  • choose Tools > Export data package
  • choose Export from the toolbar
