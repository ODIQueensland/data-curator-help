Find Menu Shortcuts

Command Description  
Find Open the find and replace panel Ctrl F ⌘ F
Find Next Find the next value after the current cell Ctrl G ⌘ G
Find Previous Find the previous value before the current cell Alt Ctrl G ⌥ ⌘ G
Replace Next Replace the next value after the current cell Ctrl E ⌘ E
Replace Previous replace the previous value before the current cell Alt Ctrl E ⌥ ⌘ E
