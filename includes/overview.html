What is Data Curator?

Data Curator helps you describe, validate and share usable open data. It is a Desktop application that performs these tasks offline, so you can prepare and check the data before sharing it online.

Data Curator follows the Frictionless Data specification to package the data, its description, and validation rules into a Data Package that can be used by the many applications and software libraries that work with Data Packages.

Using Data Curator you can:

Create, open and edit data

  • Create new data tables
  • Open data from a Separated Value File or Microsoft Excel Sheet
  • Open a Data Package from a local file or the web
  • Edit data - make changes to the values in cells, insert or remove columns or rows
  • Find and Replace data in a column

Describe data and its validation rules

  • Header Row let's you automatically set the column name property and freeze the first row when scrolling
  • Guess column properties to automatically set data type and format properties based on the data in each column
  • Set column properties to fully describe each data column and its validation rules
  • Set table properties including its primary and foreign key constraints
  • Set provenance information to help people determine if the data is fit for their purpose
  • Set Data Package properties including how it is licensed for reuse

Validate data

  • Validate a data table against each column's data type, format and constraints

Save and export data, its description and validation rules

  • Save data as a Separated Value File
  • Export a Data Package containing the data, its description and validation rules
  • Close data tables or Data Curator after saving or exporting your work

Use preferences and shortcuts to save time

  • Set preferences to specify default values
  • Menu shortcuts help you choose a menu item using the keyboard
