What is Data Curator?

Data Curator helps you describe, validate and share usable open data. It is a Desktop application that performs these tasks offline, so you can prepare and check your work before sharing it online.

Data Curator follows the Frictionless Data specification to package the data, its description, and validation rules into a Data Package that can be used by the many applications and software libraries that work with data packages. Using Data Curator you can:

Create, open and edit data

  • Create new data
  • Open a separated value file (e.g. a CSV file) or Microsoft Excel sheet
  • Open a Data Package
  • Edit data

Describe data and its validation rules

  • Guess column properties
  • Describe a data column and its validation rules
  • Describe a table
  • Describe the provenance of the data and any uncorrected errors
  • Describe the data package

Validate data

  • Validate a data table
  • Add data errors to the provenance information

Save and export data, its description and validation rules

  • Save data as a separated value file
  • Export a Data Package containing tha data and its description
  • Close data tables or Data Curator after saving or exporting your work

Use preferences and shortcuts to save time

  • Set preferences to specify default values
  • Menu shortcuts quickly choose a memu item using the keyboard
  • Editor shortcuts for when you're editing a data table
