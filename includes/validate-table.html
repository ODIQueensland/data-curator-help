After you have entered data and added Type, Format, and Contraints for each Column, you can Validate the data.

Validate table

Choose Tools > Validate, to validate the data in the current table against the Type, Format, and Contraints defined for each Column. Any validation errors will be displayed in the bottom panel.

Correct data errors

Correct the errors if you're able to. If you have a lot of errors, you can open this panel in a separate window to filter and sort the error messages.

Move the pointer over the bottom panel and click the symbol to open error messages in a separate window.

Add validation errors to Provenance Information

In some cases data errors should be corrected in the source system, and not as part of the data packaging process. If you're happy to publish the data with errors, the error messages can be appended to the provenance information so people using the data can determine if it's fit for their purpose.

Move the pointer over the bottom panel and click the symbol to add the error messages to the provenance informations.
