Open a data file

Open a tabluar data file so you can add descriptions, validation rules and more to make it easy for others to understand and use the data.

Use the Open command in the File menu to open a:

  • comma separated value file (CSV)
  • semicolon separated value file
  • tab separated value file
  • Microsoft Excel Sheet

When you open a data file, it may contain an inconsistent number of columns across the rows. This is called "ragged rows" and does not represent valid tabluar data. Ragged rows are automatically corrected when you open a data file by adding extra columns as needed.
