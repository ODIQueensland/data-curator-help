Open data file

You can open an existing data file to validate and improve it.

When you open a data file, it may contain an inconsistent number of columns across each row. This is called "ragged rows" and does not represent valid tabluar data. Ragged rows are corrected when you open a data file by adding columns as needed.

Using the menus or drag and drop, you can open a:

  • comma separated value file (CSV)
  • semicolon separated value file
  • tab separated value file
  • custom separated value file - you set the CSV Dialect
  • Microsoft Excel - you specify to open one or all Sheets

Open a comma separated value file

To open a comma separated value file, either:

  • choose File > Open > Comma separated from the menu
  • press ⌘ O

Then select a CSV file.

Open a semicolon separated value file

To open a semicolon separated value file:

  • choose File > Open > Semicolon separated from the menu
  • select a semicolon separated value file

Open a tab separated value file

To open a tab separated value (TSV) file:

  • choose File > Open > Tab separated from the menu
  • select a semicolon separated value file

Open a Custom separated value file

To open a Custom separated value file:

  • choose File > Open > Custom separated from the menu
  • specify the character(s) used to separate the values and indicate the line endings
  • select a custom separated value file

Open a Microsoft Excel file

To open a Sheet in an Excel Spreadsheet:

  • choose File > Open > Excel from the menu
  • select an Excel file
  • select All or a specific Sheet in the Excel file

Open a separated value file using Drag and Drop

To open a comma separated value file using drag and drop, select a file a drag it on to Data Curator.

To open other types of separated value files using drag and drop:

  • choose Data Curator > Preferences from the menu
  • specify the default separated value file type: Comma, Semicolon, Tab, or Custom. If you choose Custom separated, specify the CSV Dialect to use.
  • select a separated value file matching the default type and drag and drop it on to Data Curator.
