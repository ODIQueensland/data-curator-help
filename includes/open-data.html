Open a data file

Open a tabluar data file so you can add descriptions, validation rules, and more to your data. This is the first step in packaging your data to make it easy for others to understand and use with software applications, platforms and libraries.

You can open a Separated Value File or a Microsoft Excel Sheet.

Open a Separated Value File

Choose File > Open, then select the type of file, and choose the file location. You can open a:

  • Comma Separated Value file (CSV)
  • Semicolon Separated Value file
  • Tab Separated Value< file (TSV)

Your data file may contain an inconsistent number of columns across the rows. Data Curator will automatically correct this by adding extra columns as needed ensuring you have valid tabluar data.

Open a Microsoft Excel Sheet

Choose File > Open Excel Sheet, then select the file location and sheet name.

See also:

  • Set Header Row to use the values in the first row as column names. It also freezes the first row when scrolling. If you're data doesn't have headings, you'll need to add them to the first row of each column.
  • Guess Column Properties helps you quickly fill in the name (from your header row), data type, and format for each column.
  • Set Column Properties allow you enter other information about each column that can't be guessed.
  • File Menu Shortcuts can save you time by using keyboard shortcuts instead of a mouse or trackpad.
