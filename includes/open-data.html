Open a data file

Open tabular data to add descriptions and validation rules to your data. You can open a Separated Value File or a Microsoft Excel Sheet.

Open a Separated Value File

Choose File > Open, then select the type of file, and choose the file location. You can open a:

  • Comma Separated Value file (CSV)
  • Semicolon Separated Value file
  • Tab Separated Value file (TSV)

Your data file may contain an inconsistent number of columns across each row. Data Curator will automatically correct this by adding extra columns as needed ensuring you have valid tabular data.

Open a Microsoft Excel Sheet

Choose File > Open Excel Sheet, then select the file location and sheet name.

See also:

  • Edit Data to make changes to the values in cells, insert or remove columns or rows.
  • Set Header Row to use the values in the first row as column names and freeze the first row when scrolling. If you're data doesn't have headings, you'll need to add them to the first row of each column.
  • Guess Column Properties helps you quickly fill in the name (from your header row), data type, and format for each column.
  • Set Column Properties allow you enter other information about each column that can't be guessed.
  • Menu shortcuts help you choose a menu item using the keyboard
