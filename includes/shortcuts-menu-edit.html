Edit Menu Shortcuts

To do this, press...
Undo Undo the last edit command Ctrl Z ⌘ Z
Redo Repeat the last action Ctrl Y ⌘ Y
Cut Move the selected data to the clipboard Ctrl X ⌘ X
Copy Copy the selected data to the clipboard Ctrl C ⌘ C
Paste Paste the clipboard contents to the selected location Ctrl V ⌘ V
Select All Select all the data in the table Ctrl A ⌘ A
Insert Row Above Add an empty row above the current location Ctrl I ⌘ I
Insert Row Below Add an empty row below the current location Ctrl M ⌘ M
Insert Column Before Add an empty column to the left of the current location Ctrl J ⌘ J
Insert Column After Add an empty column to the right of the current location Ctrl K ⌘ K
