Menu Shortcuts

Comma Chameleon Menu

To do this, press...
Set Preferences ⌘,
Quit Comma Chameleon Ctrl + Q ⌘ Q

File Menu

To do this, press...
New ⌘ N
Open Open a Comma separated value file ⌘ O
Open Data Package Ctrl + ⇧ ⌘ P
Save Save the data in the current Tab in the default file format selected in Settings/Preferences Ctrl + S ⌘ S
Save All Save the data in all Tabs in the default file format selected in Settings/Preferences ⌥ ⌘ S
Close Tab Save the data in the current Tab and then close the Tab ⌘ W
Close Window Save the data in all Tabs and the close all Tabs ⇧ ⌘ W

Edit Menu

To do this, press...
Undo Undo the last edit command Ctrl Z ⌘ Z
Redo Ctrl Y ⌘ Y
Cut Ctrl X ⌘ X
Copy Ctrl C ⌘ C
Paste Ctrl V ⌘ V
Select All Ctrl A ⌘ A
Insert Row Above Ctrl I ⌘ I
Insert Row Below Ctrl M ⌘ M
Insert Column Left Ctrl J ⌘ J
Insert Column Right Ctrl K ⌘ K

Tools Menu

To do this, press...
Validate tbd
Column Properties tbd
Table Properties tbd
Provenance Information tbd
Data Package Properties tbd
Export Data Package tbd
Publish to GitHub tbd
Header Row Toggle the Header Row setting to freeze the first row and use the data and the Column Name property

Window Menu

To do this, press...
Show Next Tab Make the next tab the active tab Ctrl → ⌘ →
Show Previous Tab Make the previous tab the active tab Ctrl ← ⌘ ←
