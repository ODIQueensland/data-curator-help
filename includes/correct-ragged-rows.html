Correct ragged rows

When you open a data file, it may contain an inconsistent number of columns across each row. This is called "ragged rows" and does not represent valid tabluar data. If ragged rows are detected, Comma Chameleon will ask if you want these automatically corrected. Alternatively you can edit the data to manually correct it.

Once ragged rows are corrected, you'll be able to valid and share your data.

Automatically correct ragged rows

To close the data in the current Tab, either:

  • press the X symbol on the Tab
  • choose File > Close Tab from the menu
  • press Command-W

Manually correct ragged rows

To close the data in all Tabs and keep Comma Chameleon open, choose File > Close all Tabs from the menu.
