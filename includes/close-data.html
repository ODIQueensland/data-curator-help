Close data tables

When you've finished working with the data and saved your changes, you can close a data table or claose all tables.

Close a data table

After saving your work, move the pointer over the right side of a tab, click the symbol to close the table

Close Data Curator

To close all data tabs and quit Data Curator:

  • On macOS, choose Data Curator > Quit Data Curator
  • On Windows, #todo

A warning will appear stating, "There may be unsaved work. Are you sure you want to quit?". If you haven't saved your work, choose Cancel, and save each unsaved data table.

The only way to save changes you've made to the Column, Table, Provenance and Package information is to export the data package.

See also:

  • Save Data to save each data table.
  • Export Data Package to save all the data and associated Column, Table, Provenance and Package information.
