What is Data Curator?

Data Curator is a desktop CSV editor to describe, validate and share usable open data. Learn the key concepts behind Data Curator or read the guides to common tasks.

With Data Curator you can:

Create, open and edit data

  • Create new data
  • Open a separated value file (e.g. .CSV) or Microsoft Excel sheet
  • Open a Data Package
  • Edit data

Describe the data and its validation rules

  • Guess column properties
  • Describe a data column and its validation rules
  • Describe a table
  • Describe the provenance of the data and any uncorrected errors
  • Describe the data package

Validate the data

  • Validate a data table

Save and export the data

  • Save data as a separated value file
  • Export a Data Package
  • Close data tables or Data Curator

Use preferences and shortcuts to save time

  • Set preferences to specify default values
  • Menu shortcuts quickly choose a memu item using the keyboard
  • Editor shortcuts for when you're editing a data table
