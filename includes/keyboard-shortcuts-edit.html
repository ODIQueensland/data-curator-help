Editor Shortcuts

These shortcuts will help you when you're editting data

Move

To do this, press...
Move to the cell above current active cell (if exists) ↑
Move to cell underneath current active cell (if exists) ↓
move to the cell on the right side of the current active cell (if exists) →
move to the cell on the left side of current active cell (if exists) ←
Move to the cell on the right side of the current active cell (if exists) Tab
Move to the cell on the left side of current active cell (if exists) ⇧ Tab
Move to the first cell in a row Home fn →
Move to the last cell in a row End fn ←
Move to the first cell in a column Cmd + Home ⌘ fn →
Move to the last cell in a column Cmd + End ⌘ fn ←

Selection

To do this, press...
Select All
Extend selection of the cell above ⇧ ↓
Extend selection of the cell underneath ⇧ →
Extend selection of the cell on the right ⇧ ←
Extend selection of the cell on the left ⇧ ↑
select all cells in the row to the right including the current cell ⇧ Home ⇧ fn →
Select all cells in the row to the left including the current cell ⇧ End ⇧ fn ←
Select all cells in the column to the top including the current cell Ctrl ⇧ Home ⌘ ⇧ fn →
Select all cells in the column to the bottom including the current cell Ctrl ⇧ End ⌘ ⇧ fn ←

Editing

To do this, press...
insert row below (if in bottom row) Enter
insert column right (if in rightmost column) Tab
open cell editor F2
cancel editing and close cell editor Esc
empty cell Backspace or Delete
fill all selected cells with edited cell's value Ctrl Enter ⌘ Enter

Context menu

Right click in a cell to display the context menu.

To do this, press...
Move to the next option in context menu ↑
Move to the previous option in context menu ↓
Select option from context menu Enter
