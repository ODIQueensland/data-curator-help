Guess column properties

After adding or opening a data table and setting the Header Row, you can use Guess Column Properties to automatically fill in the name, data type, and format properties for each column in the data. The data type and format are used to Validate the data.

  • The name property comes from the values in the first row of date
  • The data type (e.g. string, integer, date) and format (e.g. email, dd/mm/yyyy) are inferred by reading a sample of the data in the column.

Guess Column Properties

  • Add column names to the first row if they are not already present in the data and set the Header Row
  • Choose Tools > Guess Column Properties, to automatically fill in the name, data type, and format properties for each column.
  • A panel will display to indicate the guess was successful, or display any errors.

Review the Guessed Column Properties

  • Choose Tools > Set Column Properties, to see the values that where guessed.
  • Move the cursor from column to column to see the properties set for each column.
  • Edit the data type or format if the guess was wrong.

See also:

  • Set Header Row to use the values in the first row as column names and freeze the first row when scrolling.
  • Set Column Properties allow you enter other information about each column that can't be guessed.
  • Validate Table test the data against the data type, format, and constraints.
