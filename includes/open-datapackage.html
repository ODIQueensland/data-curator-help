Open a Data Package

You can open a Tabular Data Package< to import multiple data files, their descriptions and validation rules.

Open a Data Package

Choose File > Open > Data Package, and choose the location of the Data Package from a:

  • .zip file stored at a URL
  • .zip file stored locally
  • datapackage.json file stored at a URL

On opening the Data Package:

  • each data resource is opened in a data table
  • the Column properties are set for each table
  • the Table, Provenance Information, and Data Package properties are set

If you make any changes, they cannot be saved back to the original Data Package so you will need to save your changes and export a new Data Package.

See also:

  • Save Data to save each data table.
  • Export Data Package to save all the data and associated Column, Table, Provenance and Package information.
