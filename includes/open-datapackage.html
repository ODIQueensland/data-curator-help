Open data package

You can open a data package to import multiple data files, their descriptions and validation rules. You can choose not to open the data and use the data package as a template for new data. This is useful to help a number of people create consistent data that is easy to compare and combine.

Open a data package

To open a Data Package, either:

  • choose File > Open > Data Package from the menu
  • use the shift + command + P keyboard shortcut
  • Select a datapackage.zip file and drag and drop it onto Data Curator

On opening Data Package:

  • you are asked if you want to open the data
  • each data resource is opened in a Tab
  • Column, Table, Provenance and Data Package properties are set
