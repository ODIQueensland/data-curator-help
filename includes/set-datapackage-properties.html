Set Data Package properties

When you've finished working with the data, you can save your changes and close the file using any of the steps below.

Close data in all Tabs

To close the data in all Tabs and keep Comma Chameleon open, choose File > Close all Tabs from the menu.

Close Comma Chameleon

To close all the data in all Tabs and quit Comma Chameleon, either:

  • choose Comma Chameleon > Quit Comma Chameleon from the menu.
  • press Shift+Command-W
