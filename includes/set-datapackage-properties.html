Set Data Package properties

Describe the Data Package so others can understand how they can use it. The Data Package must have a name and a licence. It is good practice to provide a title and description. Data Curator applies stricter requirements than the Frictionless Data specification to ensure you share high quality open data.

Set provenance information

  • Choose Tools > Set Data Package Properties, from the menu to display the Data Package panel
  • Enter properties to describe the Data Package
  • Data Package properties are saved when you export the data package

Required properties

  • Name - a short url-usable (and preferably human-readable) name of the package. This MUST be lower-case and contain only alphanumeric characters along with ".", "_" or "-" characters.
  • Licenses help people understand how they can use the data
    • Move the pointer over the Licence you wish to set as a default and click it to select it.
    • Hold down Ctrl/Cmd to select more than one Licence.
    • The Data Package Licence is inherited by all the tables in the package unless you explictly apply a licence to a table using Set Table Properties.

Recommended properties

  • Title - a title or one sentence description of this package.
  • Description - the first paragraph should be usable as summary information for the package.

Optional properties

  • Id - a globally unique identifiers, e.g. UUIDs and DOIs.
  • Version - follow the version pattern to indicate the nature of the change between data package releases.
  • Sources - the raw sources for this data package
    • Move the pointer over the Add source button and click it to show the data entry boxes.
    • Enter values
    • Click the Add source button again to add another source
    • Click the Delete button to remove a source
  • Contributors - the people or organizations who contributed to this Data Package.
    • Move the pointer over the Add contribitor button and click it to show the data entry boxes.
    • Enter values to set the defaults
    • Click the Add contribitor button again to add another default contributor
    • Click the Delete button to remove a default contributor
