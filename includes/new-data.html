Create new data

Using Data Curator you can create one or more related data tables, e.g.:

  • Data in a time-series, e.g. Tables of similar data for each month
  • A list of codes in one table to validate data in another table by using Foreign Keys

Create new data

When you open Data Curator, a new empty table is opened in a tab ready for you to enter data. Just start typing. Use the Tab key to move to the next column. Use the Enter key to move to the next row. There are many tools and keyboard shortcuts to help you edit the data.

Add a data table

Choose File > New, to create another data table in a new tab.

See also:

  • Edit data for tips and shortcuts to help you enter data quickly.
  • Save data to save the data as a valid separated value file.
  • Set Header Row to use the values in the first row as column names and freeze the first row when scrolling. If you're data doesn't have headings, you'll need to add them to the first row of each column.
  • Guess Column Properties helps you quickly fill in the name (from your header row), data type, and format for each column.
  • Set Column Properties allow you enter other information about each column that can't be guessed.
