Comma Chameleon is a CSV editor. You can enter data, describe, validate and share it so it's easy for everyone to use.

When you open Comma Chameleon, an empty table is conveniently waiting for your data. Just start typing.

Create new data

To create a new data table, either:

  • choose File > New from the menu
  • press ↑ N

There all lots of keyboard shortcuts to help you enter data quickly.
