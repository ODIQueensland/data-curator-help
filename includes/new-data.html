Create new data

You can create new tabular data using Data Curator. You can create more than one data table, each being shown in its own tab. You can validate data in one table against data in another table using Foreign Keys. This is the first step in packaging your data to make it easy for others to understand and use with software applications, platforms and libraries.

When you open Data Curator, a new empty table is opened in a Tab, ready for you to enter data. Just start typing. Use Tab to move to the next column. Use Enter to move to the next row. There are many tools and keyboard shortcuts to help you edit the data.

Create new data

To create a new data table, either:

  • choose File > New from the menu
  • press ⌘ N

See also:

  • Edit data for tips and shortcuts to help you enter data quickly.
  • Save data to save the data as a valid separated value file.
  • Set Header Row to use the values in the first row as column names and freeze the first row when scrolling. If you're data doesn't have headings, you'll need to add them to the first row of each column.
  • Guess Column Properties helps you quickly fill in the name (from your header row), data type, and format for each column.
  • Set Column Properties allow you enter other information about each column that can't be guessed.
