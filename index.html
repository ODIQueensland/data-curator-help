What is Comma Chameleon?

Comma Chameleon is a CSV editor makes it easy for data publishers to create, validate and share data so everyone can use it with confidence.

With Comma Chameleon you can:

  • create, open and edit tabular data
  • describe the data and its validation rules
  • validate the data
  • save and export the data, its description and validation rules
  • publish the data to GitHub and generate a web page
