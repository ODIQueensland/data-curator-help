How to add a codelist

After you create a data package, you may want to reference a codelist to:

  • provide descriptions for the codes used in the data
  • validate your data

Append data

To append data:

  • choose File > Open from the menu, new data in new tab
  • select all and copy
  • in original data tab, add row, paste
  • delete new data tab
  • update provenance information and table and datapackage properties
  • save as... new name
  • validate and export data package

Replace data

To replace data in a Data Package, either:

  • select all in orginal data tab and delete
  • choose File > Open from the menu, new data in new tab
  • select all and copy
  • paste in original data tab
  • delete new data tab
  • update provenance information and table and datapackage properties
  • save as... new name
  • validate and export data package
