How to create a data package

When creating a data package your data may come from:

  • existing data
  • new data

Use existing data to create a data package:

  • Open an existing data file
  • Validate table to check the data is tabular data
  • Set column properties to describe each column and refine the validation rules
  • Validate table to check the data complies with the column properties
  • Set table properties to describe the table
  • Set provenance information
  • Set data package properties
  • Export data package to share with others

Use existing data to create a data package:

  • Set column properties to describe each column and refine the validation rules
