How to update data in a data package

After you create a data package, you may want to update the data, with new data. Assuming you want to keep the column properties because the data structure hasn't changed, you can update the data in two ways:

  • append the data to the existing data
  • replace the data to create a new release

Append data

To append data:

  • choose File > Open from the menu, new data in new tab
  • select all and copy
  • in original data tab, add row, paste
  • delete new data tab
  • update provenance information and table and datapackage properties
  • save as... new name
  • validate and export data package

Replace data

To replace data in a Data Package, either:

  • select all in orginal data tab and delete
  • choose File > Open from the menu, new data in new tab
  • select all and copy
  • paste in original data tab
  • delete new data tab
  • update provenance information and table and datapackage properties
  • save as... new name
  • validate and export data package
