What is Comma Chameleon?

Comma Chameleon is a CSV editor makes it easy for data publishers to import, validate and share data so everyone can use it with confidence.

With Comma Chameleon you can:

  • create and import data
  • describe the data and its validation rules
  • validate the data
  • save and export the data, its description and validation rules
  • share your data on the web
