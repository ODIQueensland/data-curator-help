Open data package

You can open a data package file to import multiple data file, their descriptions and validation rules. Some data packages can be used as templates and only contain data descriptions and validation rules. This is useful to help a number of people all create consistent data.

Open a data package

To open a Data Package file, either:

  • choose File > Open > Data Package from the menu
  • use the shortcut
  • Select a data package json file and then drag and drop it onto the Comma Chameleon application

On opening Data Package:

  • data tabs opened
  • Column, Table and Data Packages properties set
  • Header row menu option set based on Table property

On opening Data Package with no data:

  • data tabs opened no header values only
  • Column, Table and Data Packages properties set
  • Header row menu option set based on Table property (should be true - test on empty data package save?)
