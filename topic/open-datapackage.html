Open data package

You can open a data package file by specifying the character(s) to separate values and indicate line endings. As the file is opened Comma Chameleon guesses the validation rules for each column. You can validate the data using these rules or refine them. Comma Chameleon also checks for ragged rows on opening the file.

Open a Custom separated file

To open a Comma separated value (CSV) file, either:

  • choose File > Open > Custom separated from the menu
  • et file preference. Select a custom file and then drag and drop it onto the Comma Chameleon application

Drag and drop a Custom separated file

To open a Custom separated value file choose File > Open > Semicolon separated from the menu

  • choose Comma Chameleon > Preferences from the menu
  • specify the character(s) used to separate values and indicate line endings.
  • select a Custom separated value file and drag and drop it onto the Comma Chameleon application
