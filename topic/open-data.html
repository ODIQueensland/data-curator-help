Open data file

You can open an existing data file to validate and improve it. As the file is opened Comma Chameleon guesses the validation rules for each column. You can validate the data using these rules or refine them. Comma Chameleon also checks for ragged rows on opening the file.

Open a Comma separated file

To open a Comma separated value (CSV) file, either:

  • choose File > Open > Comma separated from the menu
  • press Command-O
  • select a CSV file and then drag and drop it onto the Comma Chameleon application

Open a Semicolon separated file

To open a Semicolon separated value (CSV) file choose File > Open > Semicolon separated from the menu

Open a Tab separated file

To open a Tab separated value (TSV) file, choose File > Open > Tab separated from the menu

Open a Custom separated value file

You can open a non-standard separated value file by specifying the character(s) that separate values and indicate line endings. To open a Custom separated value file:

  • choose File > Open > Custom separated from the menu
  • specify the character(s) used to separate values and indicate line endings.

Open a Microsoft Excel file

To open the first Sheet in an Excel Spreadsheet, choose File > Open > Excel from the menu. Multiple sheets cannot be opened from a Spreadsheet.

Drag and drop to open a separated value file

To open a value separated value using Drag and Drop:

  • choose Comma Chameleon > Preferences from the menu
  • specify the default file type: Comma, Semicolon, Tab, or Custom separated. If you choose Custom separated, specify the character(s) used to separate values and indicate line endings.
  • select a separated value file and drag and drop it onto the Comma Chameleon application
